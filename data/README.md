# Data

This folder contains the datasets used in this project. They are **not included** in the repo due to file size limits.

Please download them manually using the instructions below.

---

## 1. RAVDESS — Emotional Speech Audio

Used for: **emotion detection / classification**

1. Go to: https://www.kaggle.com/datasets/uwrfkaggler/ravdess-emotional-speech-audio
2. Click **Download**
3. Unzip the file
4. Place the contents in `data/archive/`

Expected structure:
```
data/
└── archive/
    ├── Actor_01/
    │   ├── 03-01-01-01-01-01-01.wav
    │   └── ...
    ├── Actor_02/
    └── ...
```

---

## 2. Spotify Tracks Dataset

Used for: **music recommendation**

1. Go to: https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset
2. Click **Download**
3. Place the CSV file in `data/` and rename it to `dataset.csv`

Expected structure:
```
data/
└── dataset.csv
```

---

> 💡 You will need a free [Kaggle account](https://www.kaggle.com) to download the datasets.
